Privacy Policy

This document describes the privacy policy for the mobile game Capir (hereinafter referred to as the "Application").

What data does it collect?

The Application does not collect any personal data.

The Application collects anonymous usage information.

How does it collect data?

The Application uses ByteBrew (https://bytebrew.io/) as a service to store and retrieve usage data.

How will we use your data?

The purpose of collecting anonymous usage information is to better understand how the Application is being used. This anonymous information includes:

  • Time of use
  • Actions performed
  • Game outcome
  • Application navigation

How to contact us?

You can reach us through the email link found on the game options: game [dot] capir [at] gmail [dot] com
