Privacy Policy

This document describes the privacy policy for the mobile game Capir (hereby referred to as "Application").

What data does it collect?

The Application does not collect personal information.

The Application collects anonymous usage information.

How does it collect data

The Application uses ByteBrew (https://bytebrew.io/) as a service to store and query the usage data.

How will we use your data

The purpose of collecting anonymous usage information is to better understand how the Application is being used. This anonymous data includes:

  1. time of use
  2. actions performed
  3. game outcome
  4. application navigation
